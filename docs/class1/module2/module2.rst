Module 2: Introduction to custom NGINX Plus logging
===============================================================================

The purpose of this module is to introduce the concepts of NGINX logging with
NGINX Plus. The NGINX logging facility allows you to capture detailed information
about errors and request processing in log files. These logs can be captured
locally or sent to a syslog server.

.. note::
   Students should have a basic understanding of how to work with the CLI.
   Most importantly a desire to learn!
